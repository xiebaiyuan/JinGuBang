# Git Tools

Utilities for common Git maintenance and automation tasks.

## Tools

- `export_commits_to_patches.sh` - Export commits as patch files.
- `batch_modify_git_commits.sh` - Batch update author/commit metadata.
- `cleanup-gh-token.sh` - Cleanup GitHub token-related config/state.
- `git_so_md5.sh` - Hash helper for SO files in Git workflows.
- `list_big_repo.sh` - Inspect large repositories/objects.
- `sync_forks.sh` - Sync fork repositories.
