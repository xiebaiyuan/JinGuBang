# IP端口检测工具

## 简介

这是一个简单的Web工具，用于检测指定IPv4或IPv6地址上的端口是否开放（暴露）。通过使用纯HTML、CSS和JavaScript实现，无需服务器支持，可以在本地浏览器中运行。

## 功能特点

- 支持IPv4和IPv6地址检测
- 可同时检测多个端口
- 提供常用端口快速添加功能
- 可自定义超时时间
- 直观的检测结果展示
- 完全在前端运行，无需后端支持

## 使用方法

1. 打开 `port_checker.html` 文件（双击文件或在浏览器中打开）
2. 在"IP地址"输入框中输入要检测的IPv4或IPv6地址
3. 添加要检测的端口（可以点击"添加端口"按钮或使用常用端口快速添加功能）
4. 设置适当的超时时间（默认为2000毫秒）
5. 点击"检测端口"按钮开始检测
6. 查看检测结果

## 技术实现说明

该工具使用了以下技术:

- 使用Image对象的onload/onerror事件来检测端口是否开放
- 通过测量请求的响应时间来帮助判断端口状态
- 使用Promise和async/await处理异步操作
- 纯前端实现，无需服务器支持

## 限制说明

由于浏览器的安全限制和同源策略，此工具有一些限制:

1. 无法检测所有类型的服务，主要适用于基于HTTP/HTTPS的服务
2. 某些防火墙和网络设置可能会影响检测结果的准确性
3. 浏览器可能会阻止某些类型的请求，特别是对非标准端口的请求
4. 结果的准确性受网络环境和延迟的影响

## 注意事项

- 此工具仅用于网络故障排除和教育目的
- 请勿用于未经授权的网络扫描或渗透测试
- 频繁或大范围的端口扫描可能被视为可疑活动

## 使用场景

- 网络管理员检查服务器端口配置
- 开发人员测试应用程序的网络连接
- 排查网络连接问题
- 验证防火墙规则是否正确应用
