# Media Tools

Tools for image conversion, markdown media extraction, and metadata handling.

## Tools

- `avif_to_png_converter.py` - Convert AVIF images to PNG.
- `collect_md.py` / `collect.py` / `collect.sh` - Collect images/assets referenced by markdown.
- `modify_exif.py` / `modify_exif_demo.sh` - EXIF metadata editing examples.
- `kindle_image_converter.py` - Kindle image conversion helper.
- `apple_music_player.py` - Apple Music helper script.
- `readgeo.py` / `readgeo.ipynb` - Geo/metadata reading experiments.
- `create_test_images.py` - Generate test images.
- `setup_kindle_converter.sh` / `run_kindle_converter.sh` - Kindle converter setup/run helpers.
