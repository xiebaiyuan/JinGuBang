# File Management

Tools for organizing files, cleaning build artifacts, and analyzing duplicates.

## Tools

- `duplicate_finder.py` - Find and optionally remove duplicate files.
- `duplicate_file_analyzer.py` - Analyze duplicate files without deletion.
- `organize_files.sh` - Organize files by rules/type.
- `move_build_dirs_trash.sh` / `move_build_dirs_trash_v2.sh` / `move_build_dirs_trash_v3.sh` / `move_build_dirs_trash_v4.sh` - Move build directories to trash.
- `move_build_dirs_trash.py` - Python implementation of build directory cleanup.
- `rm_build.sh` - Build directory cleanup helper.
- `mac_cleaner.sh` - macOS cache/temp cleanup helper.
- `clean_docker.sh` - Docker cleanup helper.
